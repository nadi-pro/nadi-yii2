# Documentation

Nadi for Yii 2 is a monitoring SDK that captures exceptions, slow queries, HTTP errors,
and application performance metrics in Yii 2 applications. It supports multiple transport
drivers and configurable sampling strategies.

## Documentation Structure

### [01. Getting Started](01-getting-started/README.md)

Installation, configuration, and basic usage to get Nadi monitoring running in your Yii 2 application.

### [02. Architecture](02-architecture/README.md)

System design, core components, event handlers, and data flow through the monitoring pipeline.

### [03. Configuration](03-configuration/README.md)

Transport drivers, monitoring options, filtering rules, and sampling strategy reference.

### [04. Advanced](04-advanced/README.md)

OpenTelemetry integration, trace context propagation, and testing.
